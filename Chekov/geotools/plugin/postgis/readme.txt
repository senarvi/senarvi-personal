
GeoTools Plugin/Extention : Used extract and edit features from a PostGIS 
database.  PostGIS was developed by Refractions Research, it 
is an extension to the excellent Postgresql database, adding 
support for Geographic objects.  It is open source and 
freely available.

To use this plugin/extention ensure that gt2-postgis.jar is on your classpath.
If this directory contains a lib folder then the jars it contains must also be included.

Please report bugs and feature requests using http://jira.codehaus.org/secure/BrowseProject.jspa?id=10270.
