
GeoTools Plugin/Extention : Provides a standard interface to the DB2
Spatial Extender.  More information can be found at:
http://www.software.ibm.com/data/spatial.


To use this plugin/extention ensure that gt2-db2.jar is on your classpath.
If this directory contains a lib folder then the jars it contains must also be included.

Please report bugs and feature requests using http://jira.codehaus.org/secure/BrowseProject.jspa?id=10270.
